# 🛠️ Fun Tools International

A sleek and interactive collection of 5 fun online tools designed for entertainment and social sharing.

### ✨ Key Features:
| **Tool** | **Description** |
| :--- | :--- |
| ❤️ **Love Calculator** | Check compatibility between two names. |
| ⭐ **Kismat Calculator** | Find your destiny based on birth date. |
| 🔥 **Roast Generator** | Generate hilarious roasts for friends. |
| ✍️ **Shayari Generator** | Beautiful poetry for every mood. |
| 💬 **Tone Checker** | Analyze WhatsApp messages before sending. |

### 🚀 Tech Stack:
- **Frontend:** HTML5, CSS3 (Custom Poppins UI), Font Awesome.
- **Optimization:** SEO Friendly, Mobile Responsive, Adsterra Integrated.
